# Text-mining-on-LEOKA-data
Text mining on LEOKA data - Application of text mining on security/criminal detection
Purpose
The purpose of this text mining paper is to study a small dataset of the Law Enforcement
Officers Killed and Assaulted (LEOKA) unstructured text data collected by FBI over the
recent years. Using a simple straight-forward application of the IBM SPSS modeler text
analytics to extract text concepts using text parsing such as tokenisation by breaking down
sentences within the text into tokens, stemming to identify the root forms of the words to
avoid singular and plural forms of the same noun from forming distinctive word, creating
synonyms (Chakraborty et al., 2013), Part of Speech tagging (POS), grouping concepts and
types into explainable categories using the category rule syntax and booleans, and trying to
make sense of each categories and subcategories so that they can be transformed to actionable
categories (Zhang, 2021).
